## Projeto de Análise de Dados com Pandas - Banco

Primeiro e pequeno projeto de análise de um banco, onde foi feita a descrição da tabela, quantidade de linhas preenchidas, linhas em branco deletadas e a média de clientes em cada categoria.
Também um gráfico onde mostra a frequência de contato dos clientes com a central do banco e quantos tem solicitado cancelamento. 
É possível notar que a quantidade de cancelamentos tem sido relativamente alta nos últimos 12 meses.
